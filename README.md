# Laundry SaaS POS

A modern, AI-enhanced Point of Sale (POS) interface for a laundry business.

## Features
- Dashboard with Analytics
- Order Management (Create, View, Filter)
- Customer Management
- AI Assistant Integration
- Settings configuration (Currency, AI usage)

## Tech Stack
- Next.js (App Router)
- React
- Prisma
- TailwindCSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your environment variables in `.env` based on `.env.example` (if available)
4. Initialize the database: `npx prisma db push` (and optionally `npx prisma db seed`)
5. Run the development server: `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
